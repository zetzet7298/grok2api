🚀 SỬ DỤNG
-----------
# Test kết nối
python grok2api-client/scripts/test_connection.py

# Chat
python grok2api-client/scripts/grok_chat.py \
  --model grok-4 \
  --message "Xin chào"

# Tạo ảnh
python grok2api-client/scripts/grok_image.py \
  --prompt "Thành phố cyberpunk" \
  --n 4

# Tạo video
python grok2api-client/scripts/grok_video.py \
  --prompt "Mèo đi bộ" \
  --seconds 10

# Admin
python grok2api-client/scripts/grok_admin.py \
  --action list-tokens

📂 OUTPUT STRUCTURE
-------------------
Tất cả outputs tự động lưu vào .grok-resources/:

.grok-resources/
├── chat/          # Text responses (.txt)
├── images/        # Generated images (.png)
├── videos/        # Generated videos (.mp4)
└── metadata/      # Request/response logs (.json)
